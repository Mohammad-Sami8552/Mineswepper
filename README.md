# Minesweeper AI

This project implements an AI that plays Minesweeper using logical inference.
It was developed as part of the CS50 Artificial Intelligence course.

## Features

* Classic Minesweeper game built with Pygame
* AI agent that deduces safe moves using logical knowledge
* Browser version compiled with Pygbag

## Play Online

https://mohammad-sami8552.github.io/Minesweeper/

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the game:

python runner.py

## Build Web Version

pip install pygbag
pygbag runner.py
